Programa de Apoyo e ideas

Herramientas para estudiantes con Dislexia, Disgrafía y Discalculia

Actividades para Dislexia

Estas actividades están diseñadas para ayudar a estudiantes con dislexia a mejorar su reconocimiento de letras, sílabas y palabras.

Reconocimiento de Letras

Practica la identificación de letras con apoyo visual.

Formación de Sílabas

Combina letras para formar sílabas con retroalimentación inmediata.

Palabras Completas

Reconoce y construye palabras completas.

Ejercicio de Reconocimiento

A

Actividades para Disgrafía

Estas actividades ayudan a estudiantes con disgrafía a mejorar su escritura mediante un entrenamiento integral para las dificultades ortográficas y de integración visomotora de la disgrafia.

Copiado de letras

Practica el trazo correcto de letras con guías visuales.

Formación de Palabras

Construye palabras correctamente con apoyo visual.

Ortografía Visual

Mejora la ortografía mediante asociación visual.

Ejercicio de Escritura

Escribe la palabra: GATO

Actividades para Discalculia

Estas actividades están diseñadas para ayudar a estudiantes con discalculia a comprender conceptos matemáticos básicos mediante representaciones visuales.

Reconocimiento de Números

Identifica números y sus valores con apoyo visual.

Operaciones Básicas

Practica sumas y restas con ayudas visuales.

Problemas Visuales

Resuelve problemas matemáticos con representaciones gráficas.

Ejercicio Matemático

3 + 2 = ?

Configuración de Accesibilidad

Personaliza la experiencia de aprendizaje según las necesidades específicas del estudiante.
